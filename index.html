Installere MongoDB Community Edition på Ubuntu

Denne tutorialen viser deg hvordan du installerer MongoDB 7.0 Community Edition på Ubuntu.

Installere MongoDB

1. Importere public key

Installer gnupg og curl hvis de ikke er installert:

sudo apt-get install gnupg curl

Importer MongoDBs offentlige GPG-nøkkel:

curl -fsSL https://www.mongodb.org/static/pgp/server-7.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor

2. Lage en listefil for MongoDB

Lag listefilen /etc/apt/sources.list.d/mongodb-org-7.0.list for din Ubuntu-versjon:

echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu $(lsb_release -cs)/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list

3. Oppdatere pakkedatabasen

Oppdater den lokale pakkedatabasen:

sudo apt-get update

4. Installere MongoDB

Installer MongoDB-pakkene:

sudo apt-get install -y mongodb-org

Kjøring av MongoDB

Start MongoDB:

sudo systemctl start mongod

Sjekk at MongoDB kjører:

sudo systemctl status mongod

Sørg for at MongoDB starter automatisk ved omstart:

sudo systemctl enable mongod

Konfigurasjon

Den offisielle MongoDB-pakken inkluderer en konfigurasjonsfil /etc/mongod.conf. Etter endringer, start MongoDB på nytt:

sudo systemctl restart mongod

Avinstallere MongoDB

1. Stoppe MongoDB

Stopp MongoDB-prosessen:

sudo systemctl stop mongod

2. Fjerne Pakker

Fjern alle MongoDB-pakker:

sudo apt-get purge mongodb-org*

3. Fjerne Data Kataloger

Slett MongoDB databaser og loggfiler:

sudo rm -r /var/log/mongodb
sudo rm -r /var/lib/mongodb
